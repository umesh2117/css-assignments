Selector

1.Element Selector:
  • An element selector selects all instances of a specified HTML element on a page.
  • It uses the name of the HTML element as the selector.
  • Example: If your want ot style all pragraphs on a page, you would use the element selector 'p'.
    • p{color: blue;}

2.Id Selector:
  • An ID selector is used to select a single, unique element on a page. HTML elements can have an 'id' attribute, and the ID selector uses the value of that attribute to select the specific element.
  • The ID selector is denoted by a hash ('#') followed by the ID value.
  • Example: If you have an element with id="header", you can style it with the ID selector '#header'.
    • #header{ background-color: gray; }
3.Class Selector:
  • A class selector is used to select multiple elements that share the same class attribute.
  • The class selector is denoted by a period (.) followed by the class name.
  • Example: If you have several elements with class="highlight", you can style them with the class selector '.highlight',
    • .highlight{ fornt-weight: bold:; }
