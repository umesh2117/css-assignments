Documentation of CSS Selectors


1.Element Selector:

In CSS, an element slector targets and styles HTML elements based on their tag names.


2.Universal Selector:

In CSS, the universal sleector(*) selects and styles all elements on a webpage.


3.Group Selector:

In CSS, a group selector combines multiple selector to apply the same style rules to multiple elements.


4.ID Selector:

In CSS, an ID selector styles a specific HTML element with a unique identifier assigned using the 'id' attribute.


5.Class Selector:

In CSS, a class selector tragets and styles HTML elements with a specific class attribute.


6.Descendent Selector:

In CSS, a descendant selector selects and element that is a desendant of another specified element.


7.Child Selector:

In CSS, a child selector targets and styles elements that are direct children of a specified parent element.
